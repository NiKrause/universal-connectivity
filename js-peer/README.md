This is a [Next.js](https://nextjs.org/) project for the Universal Connectivity browser peer UI.

## Getting Started

Install dependencies with `pnpm`:

```bash
pnpm install
```

Start the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Useful commands:

```bash
pnpm lint
pnpm build
pnpm start
```
