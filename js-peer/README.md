This is a [Next.js](https://nextjs.org/) project for the Universal Connectivity browser peer UI.

## Getting Started

Install dependencies with `npm`:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Useful commands:

```bash
npm run lint
npm run build
npm run start
```
